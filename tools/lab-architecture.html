System Architecture

The Digital Intelligence Lab operates on a Zero-Backend, Client-Side Architecture. This design ensures maximum privacy for the user, as sensitive data like passwords, files, or search queries are processed locally in your browser and are never stored on our servers.

[USER BROWSER] <===> [STATIC ASSETS (GitHub/Firebase)] | v [CLIENT-SIDE LOGIC (JS/WASM)] | +-----+-----+-----+ | | | [CORS] [APIs] [LOCAL] Proxy Fetch Calc

Key Principles:

  • Stateless: No database. No tracking cookies. Every session is fresh.
  • Transparent: External API calls (HIBP, Cloudflare) are explicitly attributed.
  • Secure: File analysis (Entropy, EXIF) happens in memory using the HTML5 File API.
